# Student Course Registration System

A web application for students to register for courses, built with Node.js, Express, MongoDB, and Bootstrap.

## Features
- User Registration
- User Login
- View Available Courses
- Register for Courses
- View Registered Courses

## Technologies Used
- Node.js
- Express.js
- MongoDB
- Bootstrap 5
- HTML/CSS/JavaScript

## Deployment
Deployed on Render using three cloud service models:
- IaaS (Infrastructure as a Service)
- PaaS (Platform as a Service)
- SaaS (Software as a Service)

## Live Demo
[Render URL](https://simple-register-login-page.onrender.com)

## Local Setup
1. Clone the repository
2. Run `npm install`
3. Make sure MongoDB is running
4. Run `npm start`
5. Visit `http://localhost:3000`
